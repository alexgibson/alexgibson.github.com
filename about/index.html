I'm a front end developer living in Newcastle upon Tyne, United Kingdom. I build responsive interfaces for web applications and sites using handcrafted HTML, CSS and JavaScript.

I specialise in front end web technologies including HTML(5), CSS(3) and JavaScript. I also regularly use tools such as Sass/Compass and Jasmine in my work. When it comes to server side programming, I have some experience using PHP, MySQL and also Node.js. For version control I like to use Git, but also have experience working with both Subversion and Mercurial. My text editor of choice is Sublime Text 2.

In my spare time I help out as a co-maintainer on Mobile Boilerplate, and also write the occasional guest post for HTML5 Doctor.

I'm front end developer living in Newcastle upon Tyne, United Kingdom. I build responsive interfaces for web applications and sites using handcrafted HTML, CSS and JavaScript.

I specialise in front end web technologies including HTML(5), CSS(3), JavaScript, Sass/Compass and Jasmine. I also have some server side experience using PHP and MySQL. When it comes to version control I use Git, but also have experience with both Subversion and Mercurial. My text editor of choice is Sublime Text 2 and I generally prefer working on a Mac.

In my spare time I help out as a co-maintainer on Mobile Boilerplate, and also write the occasional guest post for HTML5 Doctor.

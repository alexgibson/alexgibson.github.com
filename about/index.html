I'm Alex Gibson, a front–end developer living in Newcastle upon Tyne, United Kingdom. My area of expertise is mainly around front end development, and technologies such as HTML, CSS and JavaScript. As well as blogging about web development here, I also maintain several open source projects over on GitHub. I also help out as a co-maintainer on Mobile Boilerplate.

Follow @alex_gibson

The site

The purpose of this site is mainly to publish my thoughts on development and serve as point of contact for projects I work on. It is by no means the best looking site in the world, but it serves its purpose of being easy to read and maintain. In terms of the technologies used, it is a simple static site built using Jekyll and is hosted on GitHub Pages. It uses Normalize.css, Sass and Compass to author the CSS. That's it.

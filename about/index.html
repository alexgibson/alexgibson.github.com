My name is Alex Gibson and I'm a freelance front-end developer living in Newcastle upon Tyne, United Kingdom. I build responsive interfaces for web applications and sites using HTML, CSS and JavaScript.

I specialise in front-end web technologies including many of the newer HTML5 JavaScript API's. I also have a good level of experience working with mobile and tablet web browsers. I can write clean, semantic HTML & CSS, and I'm just as comfortable writing vanilla JavaScript as I am picking up a popular library.

In my spare time I help out as a co-maintainer on Mobile Boilerplate, and also write the occasional guest post for HTML5 Doctor.

JavaScript Plugins

  • Concierge - A responsive UI component for installing Open Web Apps via the browser.
  • Visibility.js - A cross-browser plugin for using the Page Visibility API.
  • Notify.js - A handy wrapper for the Web Notifications API.
  • Shake.js - A custom 'shake' event plugin for mobile web browsers using device accelerometer.
  • Tap.js - A custom 'tap' event JavaScript plugin for mobile and tablet browsers.

Web applications

  • Wave-PD1 - A touch-friendly audio synthesizer, built using the Web Audio API. Also available on Chrome Web Store.
  • Checklist - A mobile web app for creating quick to-do lists. Also available on Firefox Marketplace

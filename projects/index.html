Checklist

Launch app here

Checklist is an easy to use mobile web app for creating quick to-do or shopping lists.

Built using Backbone.js and Underscore.js, Checklist is optimised for touch-based mobile & tablet web browsing, but also runs on most modern desktop browsers and is fully keyboard accessible.

  • Add, edit & delete items.
  • Batch delete items.
  • Email sharing features.
  • Completed & remaining item count.
  • Runs offline.
  • Share items via URL using: #add/your+item+here
